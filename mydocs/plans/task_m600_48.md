## char_shape start_pos 자체

body paragraph 의 char_shapes.start_pos 는 *paragraph 안 전체 utf16 position* — visible text 와 control placeholder (각 8 utf16) 포함. p.text 는 visible text 만.

p0 의 경우:
- p.text = "애국가" (3 chars)
- p.char_count = 20 (paragraph 전체 utf16)
- p.char_offsets = [16, 17, 18] (visible char 의 paragraph utf16 position)
- 즉 control placeholder 가 paragraph 자리 [0, 16) 차지, "애국가" 가 [16, 19) 차지

cell paragraph (cycle 30) 는 control placeholder 가 char_offsets 에 박히지 않아 start_pos 와 text 가 직접 매칭되지만, body paragraph 는 매칭 안 됨.

## Fix

`render_runs_with_char_shapes` 함수 자체 추가. visible char 마다 char_offsets 의 paragraph position 확인 → cs 범위 [cs_start, cs_end) 에 들어가는 visible char 만 segment 박음:

```rust
fn render_runs_with_char_shapes(p, ctx) -> String {
    let has_inline_slot = p.controls.iter().any(|c| is_hwpx_inline_slot(c));
    if p.char_shapes.len() <= 1 || has_inline_slot {
        // 단일 char_shape 또는 inline_slot control 있는 paragraph → 종전 단일 hp:run
        return format!(r#"<hp:run charPrIDRef="{}">{}</hp:run>"#, first_run_cs, render_run_content(p, ctx));
    }
    let text_chars: Vec<char> = p.text.chars().collect();
    let mut out = String::new();
    for (i, cs_ref) in p.char_shapes.iter().enumerate() {
        let cs_start = cs_ref.start_pos as usize;
        let cs_end = next.unwrap_or(p.char_count as usize);
        let mut segment = String::new();
        for (k, c) in text_chars.iter().enumerate() {
            let pos = p.char_offsets.get(k).copied().unwrap_or(0) as usize;
            if pos >= cs_start && pos < cs_end {
                segment.push(*c);
            }
        }
        // segment 가 비어도 빈 hp:run 박음 (char_shape_id 보존, cycle 34 자료)
        out.push_str(&format!(r#"<hp:run charPrIDRef="{}">{}</hp:run>"#, cs_ref.char_shape_id, t_xml));
    }
    out
}
```

조건 `has_inline_slot` — Table·Picture·Equation·Footnote 등 inline_slot control 이 들어간 paragraph 는 render_run_content 가 control_slot 을 hp:t 사이에 박는 자리라 single hp:run 으로 박아야 함. SectionDef·ColumnDef·PageNumberPos 는 hp:ctrl wrapper 박힘 → inline_slot 아님.

write_section 의 두 자리 모두 변경:
- 첫 paragraph: template 의 `<hp:run charPrIDRef="0">{first_t}</hp:run>` 자리에 `runs_xml` 박음
- 추가 paragraph (idx >= 1): `<hp:run cs={cs}>{t}</hp:run>` 자리에 `runs_xml` 박음
