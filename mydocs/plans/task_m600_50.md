# Task #m600-50 — HWPX breakLatinWord 자체 자체 attr1 bit 8 박음

브랜치 `local/task50`. cycle 49 후속 — 원본 → HWPX → HWP round-trip 자체 자체 *모든 49 ParaShape 자체 자체 자체 raw_data byte 차이* 자체 자체.

## 결함 자리

[src/parser/hwpx/header.rs](../../src/parser/hwpx/header.rs) `parse_para_shape_child` 의 `breakSetting` 분기:

```rust
b"breakNonLatinWord" => {
    // bit 7 박음
    if value == "KEEP_WORD" {
        ps.attr1 |= 1 << 7;
    } else {
        ps.attr1 &= !(1 << 7);
    }
}
// breakLatinWord 자체 자체 자체 없음
```

HWPX 의 `breakLatinWord` attribute 자체 자체 박지 않아 HWP5 ParaShape.attr1 *bit 8* 박지 못함.

원본 사업관리 참조표.hwp ps[0] attr1=0x180:
- bit 7 (0x80) = breakNonLatinWord KEEP_WORD ✅
- **bit 8 (0x100) = breakLatinWord KEEP_WORD** ❌ rhwp 자체 자체 박지 못함

round-trip 자체 자체 attr1=0x080 박혀 *bit 8 자체*.

## Fix

```rust
b"breakLatinWord" => {
    let value = attr_str(&attr);
    if value == "KEEP_WORD" {
        ps.attr1 |= 1 << 8;
    } else {
        ps.attr1 &= !(1 << 8);
    }
}
```
