# Task #m600-29 — 이중 표 (nested table) cell 편집 op 자료

## 배경

현재 cell 편집 op 4종 (`ReplaceCellRuns`·`InsertTextInCell`·`DeleteRangeInCell`·`SetCellStyle`) 은 *최상위 표 cell* 만 가리킨다. 좌표 표현이 `(section, table_para, row, col, cell_para)` 로 *단일 계층* 자료다. 표 안에 표가 들어간 *이중 표* 의 안쪽 cell 을 가리킬 방법이 없다.

원본 `1. (★사업중 필독) 사업관리 참조표.hwp` 의 3페이지 표 자료:
- `s0.p4.c0` — 외부 1x1 표 (대형 안내 박스)
- 외부 cell[0].paragraphs[8].controls[0] — nested 3x2 표 (6 cells)

cycle 28 의 시각 회복으로 *render path* 와 *직렬화 path* 는 이중 표를 정확히 박지만, *편집 path* 는 없다.

## 설계

### 자료 구조

```rust
/// 표 cell 까지의 경로. 길이 1 이면 최상위 cell, 길이 2 이상이면 nested.
/// 각 step 은 *이 step 의 표가 들어있는 paragraph 인덱스* + cell 좌표를 박는다.
/// step[i+1].para 는 step[i] 의 cell 안 paragraph 인덱스로 해석한다.
#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub struct CellPath {
    pub steps: Vec<CellStep>,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub struct CellStep {
    /// step[0] — section 안 paragraph 인덱스 (최상위 표 자리)
    /// step[i>0] — 한 단계 위 cell 안 paragraph 인덱스 (nested 표 자리)
    pub para: usize,
    /// paragraph 의 controls 안 Table control 위치
    pub ctrl_idx: usize,
    /// cell 좌표
    pub row: u16,
    pub col: u16,
}
```

### 새 EditOperation variant

기존 cell 변종 4종과 동등한 path variant 자료:

```rust
ReplaceCellRunsAtPath {
    section: usize,
    path: CellPath,
    /// 최종 cell 안 paragraph 인덱스
    cell_para: usize,
    runs: Vec<RunSpec>,
}
```

후속 작업으로 `InsertTextInCellAtPath`·`DeleteRangeInCellAtPath`·`SetCellStyleAtPath` 추가. 이 cycle 은 `ReplaceCellRunsAtPath` 자체 자체 박는다 (최소 viable 자료).

### 헬퍼

```rust
/// path 따라 재귀로 cell 위치 자체 자체 좁히고, 최종 cell 의 mutable 참조 자체 반환.
pub fn find_cell_at_path<'a>(
    doc: &'a mut Document,
    section: usize,
    path: &CellPath,
) -> Result<&'a mut Cell, HwpError>;
```

### 적용 로직

`replace_cell_runs_at_path_native` 자체 자체 자체 자체:
1. `find_cell_at_path` 로 최종 cell 참조 자체 박음
2. 기존 `replace_cell_runs_native` 의 cell 적용 로직 자체 자체 자체 자체 자체 추출·재사용
3. cell 안 paragraph[cell_para] 의 runs 자체 자체 자체 교체
4. `dirty_paragraphs` 자체 자체 자체 표시 + `paginate_if_needed`

기존 cell 변종 자체 자체 자체 자체 *path 자체 자체 자체 자체 자체* 자체 자체 자체 — 즉 `ReplaceCellRuns` 자체 자체 자체 자체 자체 자체 *path = [(table_para, ctrl_idx, row, col)]* 자체 자체 자체 *ReplaceCellRunsAtPath* 자체 자체 자체 *호출 자체 자체*. 이번 cycle 자체 자체 자체 *기존 op 자체 자체 자체 자체 자체 박지 않음*.
